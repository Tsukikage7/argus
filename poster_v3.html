核心技术亮点

ReAct 多步自主推理

不是简单的单次 LLM 问答，而是 Think → Act → Observe 闭环循环。Agent 像专家一样逐步排查，支持 Function Calling 动态调用工具。

故障回放与验证引擎

预置 级联超时 / 慢查询 / 内存泄漏 等场景，支持故障强度与流量倍率调节。一键重现故障并自动触发诊断验证。

SSE 实时全链路推送

告别黑盒诊断。Agent 每一步推理与行动都通过 Server-Sent Events 实时流式推送到前端，诊断过程完全透明可观测。

证据驱动的可解释诊断

每条根因分析与恢复建议都 回链到原始日志与 Trace 证据。输出结构化 JSON 诊断结论，支持证据包导出与历史回溯。
